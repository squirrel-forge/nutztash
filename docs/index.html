About the nutZtash

It's a lighweight todo and notes application, intended for personal use.

It can hold up to ~5mib of data in local storage, but is therefore limited in the amount of entries of varied size that can be saved at any given time.

Feel free to report bugs and suggest features, but due to previously mentioned limitations, extended features are likely not to happen.

This might only happen, if some free key-value api presents itself as an alternative storage, which would then be added with additional entry rsa encryption to ensure privacy. This could allow sharing without the need of manual imports and exports, but as of now these features are not in scope.

The privacy is all yours

This application does not use cookies or send any information across the internet.

All data, settings and theme are saved in your browsers local storage and can be imported, exported and deleted at any time.

No outside party, except what you allow within your browser (by default nobody except code on the hosting site, the application itself, or browser plugins), will be able to access any of your information.
